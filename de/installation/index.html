Überblick

Die Ghost-Dokumentation ist ein Projekt, welches ständig erneuert und verbessert wird. Wenn Sie irgendwo nicht weiterkommen oder Vorschläge für Verbesserungen haben, lassen Sie es uns bitte wissen.

Ghost basiert auf Node.js und benötigt die Version 0.10.* (letzte stabile Version).

Ghost kann problemlos auf Ihrem lokalen Computer betrieben werden, Sie müssen allerdings zuvor Node.js installiert haben.

Was ist Node.js?

Node.js ist eine moderne Plattform zur Erstellung schneller, skalierbarer und effizienter Web-Anwendungen. Über die letzten 20 Jahre hinweg hat sich das Web von einer Sammlung statischer Seite hin zu einer Plattform entwickelt, welche die Entwicklung komplexe Web-Anwendung wie Gmail und Facebook unterstützt. JavaScript ist die Programmiersprache, welche diese Weiterentwicklung ermöglicht hat.

Node.js ermöglicht es auf dem Server JavaScript zu verwenden. In der Vergangenheit existierte JavaScript nur im Browser und es wurde eine weitere Programmiersprache, zum Beispiel PHP, für die Server-seitige Programmierung benötigt. Die Möglichkeit, eine Web-Applikation mit einer einzigen Programmiersprache zu schreiben ist von grossem Vorteil und das macht Node.js für alle Entwickler interessant, die sich bisher traditionell nur auf der Client-Seite bewegt haben.

Erreicht wird dies, in dem Node.js eine Hülle um die JavaScript-Engine des Chrome-Browser von Google erstellt und diese damit portabel und überall installierbar macht. Damit können Sie Ghost sehr schnell auf Ihrem Computer installieren und einfach ausprobieren. Die folgenden Abschnitte zeigen im Detail wie Sie Ghost lokal auf Ihrem Computer mit Mac, Windows oder Linux oder alternativ auf einem Server oder Hosting-Dienst installieren können.
