 Download Node.js

Ghost requires node 0.10.* or 0.11.* and npm. To get these you go to http://nodes.org and download your operating system's version of Node.js.

Windows

Click on the NodeJS .exe and proceed with the install.

 Mac

Click on the NodeJS .dmg and proceed with the install.

 Install NPM

  1. Open Terminal / Command Prompt
  2. Navigate to your Ghost directory
  3. Then enter npm install

 Setting Up Ghost

 Installing Ghost

  1. In the Terminal / Command Prompt, type npm start

  2. This will have launched your Ghost blog, visit one of the folowing URLs to see;

  3. Development: http://localhost:2368/

  4. Production: Your URL
